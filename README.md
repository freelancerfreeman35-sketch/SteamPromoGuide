#SteamMarketingPlaybook
A collection of strategies, guides, and tools for marketing and promoting games on Steam. Focused on helping indie developers improve visibility, increase wishlists, and grow sales.
## What you'll learn
- How to get more wishlists
- How Steam's algorithm works
- Store page optimization
- Launch strategies
## Who this is for
Indie game developers and marketers looking to grow their audience on Steam.
